# Video Slicer PE

Batch-скрипт для Windows: режет запись стрима на короткие клипы для заполнения профилей моделей. Для операторов/менеджеров моделей.

## Что делает

Для каждого видео из `videos\`:

- Нарезает на клипы **18 сек** и **1:05**, в **16:9** и **9:16**
- Делает версии с оригинальным звуком, без звука (тизеры) и с треком из `music\`
- Снимает стоп-кадры каждые 3 сек (x2 upscale)

Несколько видео обрабатываются параллельно.

## Требования

- Windows 10/11
- FFmpeg — системный или в папке `ffmpeg\`

## Как запустить

1. Положи `ffmpeg.exe` и `ffprobe.exe` в папку `ffmpeg\` (взять из любой готовой портативной сборки ffmpeg или скачать: https://github.com/BtbN/FFmpeg-Builds/releases). Если ffmpeg уже установлен в систему — этот шаг не нужен, скрипт найдёт его сам
2. Положи нарезки стрима (`.mp4`, `.mov`, `.mkv`) в `videos\`
3. (Опционально) Положи в `music\` треки, разрешённые площадкой — скрипт накладывает их вместо оригинального звука
4. Запусти `AutoAllPE.bat`
5. Жди "✅ ВСЁ ГОТОВО!"

## Результат

- `clips\` — нарезки с оригинальным звуком
- `JAS\` — нарезки с разрешённым треком
- `JAS_teaser\` — нарезки без звука
- `frames\` — стоп-кадры

## Музыка

В `music\` класть только треки, разрешённые конкретной площадкой для использования в профиле — иначе ролик не пройдёт модерацию или прилетит страйк. Произвольную музыку из интернета сюда не класть.
